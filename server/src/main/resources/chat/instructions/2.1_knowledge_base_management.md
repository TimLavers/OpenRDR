## Background
- The application has a set of Knowledge Bases that the user can work on.
- The user can only work on one Knowledge Base at a time.

## Tasks

### Getting a list of Knowledge Bases
If the user indicates that they would like to see what Knowledge Bases are available, output this response:
{
"action": "ListKnowledgeBases"
}

### Selecting a Knowledge Base
If the user indicates that they would like to open a particular Knowledge Base, return this response:
{
"action": "OpenKnowledgeBase",
"kbName": "<name of the Knowledge Base that the user would like to open>"
}
``
